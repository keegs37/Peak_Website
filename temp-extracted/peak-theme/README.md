# PEAK Internet Custom WordPress Theme

This theme is generated from the PEAK Internet HTML files.

## Installation Instructions

1.  **Zip the Theme Directory:**
    Compress the entire `peak-theme` directory into a `.zip` file (e.g., `peak-theme.zip`).
2.  **Upload to WordPress:**
    Log in to your WordPress dashboard. Navigate to **Appearance > Themes**.
    Click "Add New" at the top, and then "Upload Theme".
    Choose the `peak-theme.zip` file you created and click "Install Now".
3.  **Activate the Theme:**
    Once installed, click the "Activate" link.
4.  **Create Pages:**
    Create pages in WordPress corresponding to the HTML pages originally in the site. 
    Make sure to select the correct Page Template for each page under "Page Attributes" on the right sidebar while editing a page. For example, for the Internet Plans page, select the "Internet Plans" template.
5.  **Set Front Page (Homepage):**
    Go to **Settings > Reading**. Choose "A static page" for your homepage displays setting. Select your homepage from the dropdown (you will need to create a page for the Home first). Your homepage will automatically use the `front-page.php` template.
6.  **Navigation Menus:**
    This theme supports the following menu locations:
    - Primary Menu
    - Footer Services
    - Footer Support
    - Footer Company
    You can create and assign menus to these locations by going to **Appearance > Menus**.
